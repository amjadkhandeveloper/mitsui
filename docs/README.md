# Documentation

This folder contains detailed documentation for each screen/feature in the Mitsui app.

## Structure

Each screen/feature has its own folder containing:
- `screen_details.md` - Screen specifications, UI/UX details, user flows
- `api_specs.md` - API endpoints, request/response formats, error handling
- `design_assets.md` - Design references, images, color schemes
- `implementation_notes.md` - Technical implementation details, decisions
